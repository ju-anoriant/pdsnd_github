# Bikeshare stats
Simple script to get insights about bike share data.

## Requirements
* Python >= 3.7
* Numpy >= 1.17
* Pandas >= 0.25

## Usage
* clone the repository
* run the script: \
`python bikeshare.py`

## Files used
- bikeshare.py
- chicago.csv
- new_york_city.csv
- washington.csv
